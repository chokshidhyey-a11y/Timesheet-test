## Deploy to Vercel (Free — recommended)

1. Go to https://vercel.com and sign up (free)
2. Click **"Add New Project"**
3. Click **"Upload"** and drag this entire folder in
4. Click **Deploy**
5. You'll get a live URL like `eurospec-timesheet.vercel.app` in ~60 seconds

---

## Deploy to Netlify (Also free)

1. Go to https://netlify.com and sign up (free)
2. Drag this entire folder onto the Netlify dashboard
3. Done — you'll get a live URL instantly

---

## Run locally (if you have Node.js installed)

```bash
npm install
npm start
```
Opens at http://localhost:3000
